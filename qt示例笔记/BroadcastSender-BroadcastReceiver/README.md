## BroadcastReceiver

### 示例功能

创建UDP连接，监听本地`45454`端口，定有广播数据时接收并显示在QLabel上。

### 代码分析

本示例的代码比较简单，创建一个UDP连接，当有数据可以读取时，QUdpSocket会发射readyRead()信号，此时可以调用槽函数来接收数据。完整代码见下图
